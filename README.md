# Fuzzy Inference System for Yakiniku Like Inventory Prediction

## Project Overview
This project aims to develop a Fuzzy Inference System (FIS) to predict inventory needs for the Yakiniku Like restaurant chain. By utilizing fuzzy logic, we can effectively handle the uncertainties and complexities associated with inventory management.

## Objectives
- To accurately predict the amount of inventory required for various Yakiniku dishes.
- To optimize supply chain operations and reduce waste by improving inventory forecasting.

## Methodology
1. **Data Collection**: Gather historical data on inventory levels, sales, and customer demand.
2. **Fuzzy Logic Design**: Define fuzzy sets and rules that model the relationship between input variables (e.g., historical sales, seasonal trends) and output variables (e.g., inventory levels).
3. **System Implementation**: Develop the FIS using a programming language and libraries suitable for fuzzy logic operations.
4. **Testing and Validation**: Validate the performance of the FIS against real historical data to ensure accuracy in predictions.

## Expected Outcomes
- Improved accuracy in inventory predictions.
- Enhanced decision-making processes regarding stock management.
- Reduction in operational costs through better inventory control.

## Conclusion
The Fuzzy Inference System for Yakiniku Like inventory prediction represents a significant step towards leveraging advanced computational techniques to improve restaurant supply chain efficiency. By implementing this system, we aim to enhance customer satisfaction while minimizing food waste and operational costs.
